# AI Görsel Üretim Promptları - Kategoriler

Bu dosya, her kategori için yüksek kaliteli görsel üretmek için hazırlanmış AI promptlarını içerir. 
Görseller modern, minimal, uygulama tasarımına uygun ve profesyonel olmalıdır.

## Genel Gereksinimler
- **Stil**: Modern, minimal, flat design, gradient kullanımı
- **Renkler**: Her kategorinin kendi renk paleti (yukarıda belirtilen)
- **Format**: 1:1 kare format (1024x1024 veya 512x512)
- **Stil**: İllüstrasyon, vektör tarzı, profesyonel
- **Arka Plan**: Gradient veya solid renk, temiz ve modern

## Kullanım Notları

1. **AI Tool Önerileri**: 
   - Midjourney
   - DALL-E 3
   - Stable Diffusion
   - Adobe Firefly

2. **Görsel Boyutları**:
   - Ana görsel: 1024x1024 px (veya 512x512)
   - Format: PNG (şeffaf arka plan tercih edilir)
   - Çözünürlük: Yüksek kalite (300 DPI)

3. **Stil Tutarlılığı**:
   - Tüm görseller aynı stil dilini kullanmalı
   - Renk paletleri tutarlı olmalı
   - Minimal ve modern yaklaşım korunmalı

4. **Optimizasyon**:
   - Görselleri uygulama için optimize edin
   - Dosya boyutunu küçültün (WebP formatı önerilir)
   - Farklı ekran boyutları için responsive versiyonlar oluşturun

5. **Test**:
   - Görselleri uygulamada test edin
   - Farklı arka plan renklerinde görünümünü kontrol edin
   - Kullanıcı geri bildirimlerine göre ince ayar yapın

## Alternatif Prompt Stilleri

Eğer yukarıdaki promptlar istediğiniz sonucu vermezse, şu alternatif yaklaşımları deneyebilirsiniz:

### Stil 1: Daha Detaylı
```
[Kategori adı] için modern, profesyonel, minimal flat design illustration. [Renk] gradient arka plan, temiz ve modern tasarım. Vektör sanat stili, mobil uygulama için uygun. Metin yok, sadece görsel öğeler. Yumuşak gölgeler, yuvarlatılmış köşeler, [kategori] temasına uygun atmosfer.
```

### Stil 2: Daha Basit
```
Minimalist [kategori] icon, flat design, [renk] gradient background, clean vector art, mobile app style, no text.
```

### Stil 3: Daha Yaratıcı
```
Modern illustration of [kategori elementi], vibrant [renk] colors, minimalist style, professional design, suitable for social event app, clean and inviting.
```
